## 1. 产品简介

### 1.1 产品定位

电商合规哨兵是一款基于AI技术的电商合规风险管理平台,专门为电商企业、跨境电商和平台商家打造。通过智能化的主从Agent架构,提供实时的合规风险监控、预警和分析服务,帮助企业规避违规风险,降低合规成本。

### 1.2 核心功能

- **风险预警**: 7×24小时监控电商法规变化,提前预警风险,响应时间<5分钟
- **风险识别**: AI驱动智能分析商品描述、广告文案、促销活动中的合规风险,准确率>85%
- **罚款预测**: 基于历史案例和法规数据库,预测潜在罚款金额和风险概率
- **知识图谱查询**: 覆盖10万+电商法规节点,快速检索相关法规和案例
- **智能整改建议**: 自动生成具体的整改方案,降低人力成本70%以上
- **报告导出**: 生成专业的合规报告,支持PDF、Word、Excel等多种格式

### 1.3 适用场景

- **日常合规检查**: 定期检查商品描述、宣传文案是否符合法规要求
- **促销活动审核**: 双11、618等重大促销活动前的合规风险评估
- **新品上线预审**: 新产品发布前的合规风险预审
- **跨境业务合规**: 跨境电商的多国法规合规评估
- **监管检查应对**: 提供完整的风险处理记录,应对监管检查

## 2. 快速开始

### 2.1 注册账号

**步骤1**: 访问官网
- 打开浏览器,访问电商合规哨兵官网
- 点击页面右上角的"免费注册"按钮

**步骤2**: 填写注册信息
- 输入企业名称、联系人姓名、手机号码
- 设置登录密码(建议8-16位,包含字母和数字)
- 输入短信验证码

**步骤3**: 完成注册
- 阅读并同意《用户服务协议》
- 点击"立即注册"按钮
- 系统自动创建账号并跳转到登录页面

**界面说明**:
- 注册页面顶部显示产品Logo和品牌名称
- 表单字段右侧有必填标识(*)
- 手机号验证码有效期为5分钟

### 2.2 创建项目

**步骤1**: 登录系统
- 输入手机号和密码登录系统
- 首次登录会进入新手引导页面

**步骤2**: 创建项目
- 点击首页的"创建项目"按钮
- 填写项目基本信息:
  - 项目名称: 例如"XX旗舰店合规管理"
  - 项目类型: 选择电商平台类型(天猫/京东/拼多多等)
  - 店铺名称: 填写实际店铺名称
  - 店铺链接: 填写店铺主页链接

**步骤3**: 配置监控范围
- 选择需要监控的商品类目
- 设置监控频率(实时/每日/每周)
- 配置预警阈值(高风险/中风险/低风险)

**步骤4**: 完成创建
- 点击"创建项目"按钮
- 系统自动初始化项目,预计1-2分钟

**界面说明**:
- 项目创建页面左侧为导航栏,显示创建步骤进度
- 右侧为表单区域,支持快速填写
- 底部有"上一步""下一步""保存草稿"等操作按钮

### 2.3 配置平台信息

**步骤1**: 进入项目设置
- 在项目列表中点击目标项目
- 点击右上角"项目设置"按钮

**步骤2**: 配置平台API
- 选择电商平台类型
- 输入平台API密钥和密钥密码
- 点击"验证连接"按钮测试连接

**步骤3**: 配置通知方式
- 选择通知渠道: 飞书/钉钉/企业微信/邮件/短信
- 输入Webhook地址或接收邮箱
- 设置通知频率: 实时推送/每日汇总/每周汇总

**步骤4**: 保存配置
- 点击"保存配置"按钮
- 系统自动验证配置有效性

**界面说明**:
- 平台配置页面采用卡片式布局
- 每个平台配置独立为一个卡片
- 支持同时配置多个电商平台

## 3. 功能说明

### 3.1 风险预警功能

**功能概述**: 实时监控电商法规变化和商品风险,通过多渠道及时推送预警信息。

**操作步骤**:
1. 进入"风险预警"模块
2. 查看预警列表(显示风险等级、风险类型、发现时间)
3. 点击预警详情查看具体风险描述
4. 查看相关法规条款和相似案例
5. 一键生成整改建议

**预警等级**:
- 🔴 **高风险**: 可能导致重大罚款或店铺关停,需立即处理
- 🟡 **中风险**: 存在违规风险,建议尽快整改
- 🟢 **低风险**: 轻微问题,可定期复查

**界面说明**:
- 预警列表支持按等级、类型、时间筛选
- 点击预警条目右侧的"详情"按钮展开详细信息
- 详情页显示风险描述、法规依据、整改建议、相似案例

### 3.2 罚款预测功能

**功能概述**: 基于AI模型和历史案例,预测潜在罚款金额和风险概率。

**操作步骤**:
1. 进入"罚款预测"模块
2. 选择需要分析的商品或文案
3. 系统自动分析风险因素
4. 查看预测结果:
   - 预测罚款金额范围
   - 风险概率百分比
   - 主要风险因素分析
5. 查看相似违规案例及处罚结果

**预测指标**:
- **罚款金额**: 基于历史案例计算的罚款范围
- **风险概率**: 发生违规的概率百分比
- **影响范围**: 可能影响的商品数量和用户群体

**界面说明**:
- 预测结果以可视化图表展示
- 支持导出预测报告(PDF格式)
- 可对比不同商品的风险预测结果

### 3.3 知识图谱查询

**功能概述**: 提供10万+电商法规节点的知识图谱,支持智能检索和关联分析。

**操作步骤**:
1. 进入"知识库"模块
2. 在搜索框输入关键词(如"虚假宣传""价格欺诈")
3. 系统展示相关法规、案例、解读文章
4. 点击法规条款查看详细内容
5. 查看相关法规的关联图谱

**知识库内容**:
- 电商法规数据库: 覆盖20+国家/地区的法规
- 违规案例库: 5万+真实违规案例
- 合规整改方案库: 历史整改成功案例
- 行业标准库: 各行业合规标准

**界面说明**:
- 搜索结果按相关度排序
- 支持高级搜索(按时间、类型、关键词组合)
- 法规详情页显示关联法规和相关案例

### 3.4 应对建议生成

**功能概述**: AI自动生成具体的整改建议,降低合规人力成本70%以上。

**操作步骤**:
1. 在风险预警详情页点击"生成建议"
2. 系统AI分析风险因素
3. 生成整改建议(包含具体操作步骤)
4. 查看建议详情:
   - 整改方案
   - 预期效果
   - 参考案例
5. 一键应用建议或导出建议文档

**建议内容**:
- **问题诊断**: 明确指出违规点
- **整改方案**: 提供具体的整改步骤
- **合规文案**: 生成符合规范的替代文案
- **风险评估**: 整改后的风险评估

**界面说明**:
- 建议页面采用步骤式布局
- 每个建议包含详细说明和操作指南
- 支持收藏、分享、导出建议

### 3.5 报告导出

**功能概述**: 生成专业的合规分析报告,支持多种格式导出。

**操作步骤**:
1. 进入"报告中心"模块
2. 选择报告类型:
   - 风险分析报告
   - 合规检查报告
   - 整改报告
   - 综合报告
3. 设置报告参数(时间范围、分析维度)
4. 点击"生成报告"
5. 选择导出格式(PDF/Word/Excel)
6. 下载报告文件

**报告内容**:
- **执行摘要**: 风险概况和关键发现
- **详细分析**: 各维度风险分析
- **数据可视化**: 图表和趋势分析
- **整改建议**: 整改方案和行动计划
- **附录**: 详细数据表和法规依据

**界面说明**:
- 报告列表显示历史生成的报告
- 支持报告预览功能
- 报告模板可自定义

A: 完善商品信息和店铺配置,设置合理的监控范围,定期更新平台API配置。

### 6.1 账号相关

**Q: 忘记密码怎么办?**
A: 点击登录页面的"忘记密码",通过手机验证码重置密码。
